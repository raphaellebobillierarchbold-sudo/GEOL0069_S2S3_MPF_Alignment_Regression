# Raw data

The raw `.npz` files are not included in this GitHub repository because they are too large for standard GitHub upload.

To reproduce the project, place the following files in this folder:

```text
data/raw/
├── s2_zoomed_data.npz
├── s3_zoomed_data.npz
├── mpf_zoomed_data.npz
├── interpolated_data.npz
└── training_data_subsubarea_noalign.npz
```

These files are derived from the GEOL0069 Week 6 Sentinel-2/Sentinel-3 alignment and MPF regression practical.
