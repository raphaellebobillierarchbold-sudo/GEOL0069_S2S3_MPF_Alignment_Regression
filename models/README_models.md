# Model files

Trained models are generated by `notebooks/03_regression_models.ipynb`.

Small model artefacts may be committed to GitHub. Large files, especially Random Forest `.joblib` models, can be stored externally or regenerated by rerunning the notebooks.

Expected local model files:

- `polynomial_aligned.joblib`
- `polynomial_noalign.joblib`
- `neural_network_aligned.keras`
- `neural_network_noalign.keras`
- `neural_network_scaler_aligned.joblib`
- `neural_network_scaler_noalign.joblib`
- `gaussian_process_aligned.joblib`
- `gaussian_process_noalign.joblib`

Random Forest model files are large and may be omitted from GitHub:

- `random_forest_aligned.joblib`
- `random_forest_noalign.joblib`

All model predictions can be reproduced by running Notebook 3.
