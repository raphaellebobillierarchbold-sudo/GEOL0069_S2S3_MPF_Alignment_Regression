
# Model files

This folder contains trained model artefacts generated by `notebooks/03_regression_models.ipynb`.

Some trained model files are not included directly in this GitHub repository because they are large binary files and exceed the practical upload limit for GitHub browser uploads.

The full model outputs can be regenerated by running:

1. `notebooks/02_create_aligned_misaligned_datasets.ipynb`
2. `notebooks/03_regression_models.ipynb`

Expected generated files:

- `polynomial_aligned.joblib`
- `polynomial_noalign.joblib`
- `random_forest_aligned.joblib`
- `random_forest_noalign.joblib`
- `neural_network_aligned.keras`
- `neural_network_noalign.keras`
- `gaussian_process_aligned.joblib`
- `gaussian_process_noalign.joblib`

The large Random Forest model files are stored externally on Google Drive for reproducibility:

- `random_forest_aligned.joblib`
- `random_forest_noalign.joblib`

Google Drive model folder: https://drive.google.com/drive/folders/1sSHi-GnsSM6QViwQMOZEOIJynSatQ5pB?usp=drive_link
